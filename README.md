## Description
A point and click guessing game. With nine viable options to choose 
from. Shows a fluffy jiglypuff pokemon as a correct answer with
appropriate response and a wrong image vis-a-vis to incorrect answer.

## Build with
	- React
	- Tailwindcss
	- Framer Motion
	- React Icons
